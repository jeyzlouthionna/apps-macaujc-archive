# apps-macaujc-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内容涵盖不同主题的静态页面资源，以方便浏览和引用。本仓库不针对任何特定域名或网站。

## 项目简介

本仓库的主要目的是提供一个集中存放 HTML 页面副本的空间。这些页面可能来源于不同时期或不同用途，经过整理后统一归档于此，便于后续查阅或分发。

## 目录结构

```
.
├── pages/          # 存放独立 HTML 页面的目录
├── assets/         # 页面引用的静态资源（CSS、图片、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立的页面，文件名通常反映页面主题或归档日期。
- `assets/`：存放页面所需的公共或私有资源文件，按需组织。

## 页面归档说明

- 所有页面均为静态 HTML 文件，可直接在浏览器中打开。
- 页面可能包含外部资源引用，但已尽可能内联或本地化，减少外部依赖。
- 归档文件可能来自不同时间点，内容可能未及时更新，请以实际展示为准。
- 本仓库不保证所有页面长期可用或兼容所有浏览器版本。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已知问题。
- 欢迎提交 Issue 或 Pull Request 协助改进归档内容。
- 对于明显过时或错误的页面，维护者可能会移除或替换。
- 请勿将本仓库用于任何违反法律法规或侵犯他人权益的目的。

## 使用方式

您可以直接通过 GitHub 页面浏览或下载整个仓库。如需在本地查看，请克隆仓库后使用任意静态文件服务器（如 Python 的 `http.server`）运行。

```bash
git clone https://github.com/your-username/apps-macaujc-archive.git
cd apps-macaujc-archive
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/pages/` 查看页面列表。
